💰 2️⃣ Expense Tracker – JavaScript
📌 Description

A sleek and interactive expense tracker that allows users to add, manage, and calculate daily expenses. The application dynamically updates the total amount using JavaScript and demonstrates real-time state management without any external libraries.

🚀 Features

Add and delete expenses

Automatic total calculation

Dynamic list rendering

Clean dark modern UI

Form validation

🛠 Tech Stack

HTML5

CSS3

JavaScript (ES6)

📚 Concepts Used

Array methods (map, reduce, forEach)

Event Listeners

DOM manipulation

State management using arrays

Conditional UI rendering
