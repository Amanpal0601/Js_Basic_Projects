🌤 1️⃣ Weather App – JavaScript

📌 Description

A modern weather application built using HTML, CSS, and JavaScript that fetches real-time weather data from a public API. The app dynamically updates UI components based on user input and displays temperature, city name, and weather conditions with a clean animated interface.

🚀 Features

Real-time API integration
Dynamic DOM manipulation
Error handling for invalid city names
Animated gradient background
Responsive glassmorphism UI

🛠 Tech Stack

HTML
CSS (Glassmorphism + Animations)
JavaScript
Weather API

📚 Concepts Used

Fetch API
Async/Await
DOM Manipulation
Event Handling
Conditional Rendering
